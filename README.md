# Projeto Safira

## Protótipo de carro autônomo

## Execução

Para executar o projeto é nescessário possuir no ambiente:

 - Python
 - PiEnv
 - OpenCV v3
 
 
## IDE

Recomendado a utilização da IDE Pycharm - Jetbrains

## LOG
